Guttenberg-model
================

From:
Guttenberg, N., & Goldenfeld, N. (2008). Cascade of complexity in evolving predator-prey dynamics. Physical review letters, 100(5), 058102.

Python implementation by Norman Packard

Aug 2014

Executable:  gutt.py

brings up the 2d world in a pygame window.

Species information,s printed on stdout, suitable for graphing by activity.py, e.g.

activity.py gutt.py

To run without activity:

gutt.py > /dev/null
